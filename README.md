# api-hub
GraphQL-based API server to encapsulate all kinds of REST APIs.

## Usage
```bash
$ git clone https://github.com/zhicwu/api-hub.git
$ cd api-hub
$ npm i
$ npm start
```
Now open your browser to access http://localhost:4000/graphql
